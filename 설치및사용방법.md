# 📘 Triple 번역 & 요약 프로그램 설치 및 사용 설명서

본 프로그램은 내 컴퓨터에 설치된 로컬 AI(Ollama)를 활용하여 영문 텍스트를 한국어로 번역하고, 번역된 내용을 요약하여 깔끔한 노트 형식(마크다운 파일)으로 저장해 주는 편리한 데스크톱 도구입니다.

컴퓨터 초심자분들도 쉽게 시작할 수 있도록 단계별 가이드를 작성했습니다. 차근차근 따라 해 보세요!

---

## 🛠️ 1단계: 파이썬(Python) 설치하기 (최초 1회)

프로그램을 구동하기 위한 기본 엔진인 파이썬을 설치해야 합니다.

1. **파이썬 다운로드 페이지 접속**: 인터넷 브라우저를 열고 [python.org](https://www.python.org/downloads/) 에 접속합니다.
2. **다운로드**: 노란색 **`Download Python 3.xx.x`** 버튼을 클릭하여 설치 파일을 다운로드합니다.
3. **설치 시작**: 다운로드한 설치 파일(예: `python-3.xx.x-amd64.exe`)을 실행합니다.
4. **⚠️ 가장 중요 (별표 5개)**:
   - 설치 창이 뜨면 가장 하단에 있는 **`[ ] Add python.exe to PATH`** (파이썬을 환경 변수에 추가) 체크 박스를 **반드시 체크**해 주세요! 이 부분을 체크하지 않으면 프로그램이 실행되지 않습니다.
5. **설치 완료**: **`Install Now`** 버튼을 눌러 설치를 완료합니다.

---

## 🤖 2단계: Ollama 및 AI 모델 준비하기

로컬 AI 모델을 내 컴퓨터에서 실행하기 위한 단계입니다.

1. **Ollama 다운로드 페이지 접속**: [ollama.com](https://ollama.com/) 에 접속합니다.
2. **설치**: **`Download`** 버튼을 눌러 내 컴퓨터 운영체제(Windows)에 맞는 설치 파일을 받아 실행하고 설치를 진행합니다. (설치가 끝나면 화면 우측 하단 작업 표시줄에 라마 모양 아이콘이 나타납니다.)
3. **AI 모델 다운로드**:
   - 키보드의 `시작 버튼`을 누르고 **`cmd`** 또는 **`명령 프롬프트`**를 검색하여 실행합니다.
   - 검은색 터미널 창이 뜨면 아래 명령어를 입력하고 키보드의 `Enter`를 누릅니다:
     ```bash
     ollama run hf.co/lmstudio-community/gemma-4-E4B-it-GGUF:Q4_K_M
     ```
   - 모델 다운로드가 자동으로 진행됩니다. 100% 완료된 후 다운로드가 끝나면 터미널 창을 닫아도 됩니다.

## 📂 3단계: 프로그램 파일 복사 및 실행하기

1. **프로그램 다운로드**: 깃허브(GitHub) 저장소에서 초록색 **`Code`** 버튼을 누른 뒤 **`Download ZIP`**을 클릭하여 프로그램 소스 파일을 다운로드합니다.
2. **압축 풀기**: 다운로드한 ZIP 파일의 압축을 해제합니다. **위치는 컴퓨터 내 원하는 곳 어디든(바탕화면, 문서 폴더 등) 상관없습니다.**
3. **프로그램 실행**:
   - 압축을 해제한 폴더로 들어갑니다.
   - 폴더 내에 있는 **`run.bat`** (톱니바퀴 아이콘 모양 배치 파일)을 더블클릭합니다.
   - 최초 실행 시 필요한 데스크톱 라이브러리(`pywebview`)를 알아서 자동으로 설치한 뒤 프로그램 창이 나타납니다.

## 💻 4단계: 프로그램 사용 방법

프로그램 화면은 크게 3개의 탭(**원문**, **번역문**, **요약문**)으로 구성되어 있습니다.

### ① 텍스트 입력 및 로드
* **원문** 창에 번역하고자 하는 영어 텍스트를 직접 복사해서 붙여넣거나, 각 창 상단의 파일 불러오기 버튼(`📂`)을 통해 기존 저장한 파일을 불러올 수 있습니다.
* 개별 탭에서 파일을 하나씩 불러오더라도 이미 입력해 둔 다른 탭의 데이터는 사라지지 않고 유지됩니다.

### ② AI 전체 번역 및 요약 실행
* **전체 번역**: 원문 창 상단의 **`🌐 전체 번역`** 버튼을 누르면 AI가 각 문단을 한국어로 번역하여 번역문 창에 채워 넣습니다.
* **전체 요약**: 번역문 창 상단의 **`📝 전체 요약`** 버튼을 누르면 번역된 한글 내용을 토대로 최종 요약문이 생성됩니다.
  * 💡 **꿀팁 (요약 제외 기능)**: 특정 문단의 요약이 불필요한 경우(예: 제목 등), 요약문 셀에 직접 **`제외`**, **`요약 제외`**, 또는 **`요약제외`**라고 입력해 두면 AI 요약을 거치지 않고 왼쪽 번역문의 내용이 그대로 적용됩니다.

### ③ 결과 저장하기
* **MD 저장 (옵시디언 호환)**: 요약문 창 우측 상단의 **`MD`** 버튼을 누르면 마크다운 파일(`.md`) 형식으로 저장됩니다.
  * 저장된 파일은 요약문 본문이 바로 보이고, 번역문과 원문은 클릭해야 펼쳐지는 깔끔한 **접기 상자(`<details>`)** 서식으로 저장되어 독서 및 공부용 노트로 활용하기 최적화되어 있습니다.
* **전체 저장 & Triple 불러오기**:
  * 상단 헤더의 **`💾 전체 저장`** 버튼을 누르면 원문, 번역문, 요약문 텍스트 전부가 하나의 파일로 백업됩니다.
  * 나중에 다시 작업을 이어서 하고 싶다면 앱 최우측 상단의 **`📂 Triple 불러오기`** 버튼을 눌러 백업 파일을 불러오기만 하면 문단 매칭 상태 그대로 원상 복원됩니다.
